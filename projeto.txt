Crie um software que permita que o usuário entre com um texto, o idioma em que o texto 
está escrito e também o idioma para qual ele deseja traduzir o texto e receba como resposta na tela o texto traduzido.


Critérios:

    Você deve utilizar orientação a objetos no seu software
    O resultado de cada tradução (junto com a frase original) deve ser guardado em um arquivo que possua no 
    nome a data e horário da tradução (exp: 10-10-18_10:30.txt)

1- Pedir para o usuário digitar o texto, idioma de entrada e saida;
2- levar os dados para a classe de tradução;
3- fazer a chamada web para a api do Yandex;
4- receber a resposta e printar para o usuário.
